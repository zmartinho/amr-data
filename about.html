About This Project

This website provides an interactive platform for exploring global data on antimicrobial resistance (AMR). It allows users to view and compare data from different microorganisms across countries using both map and chart visualizations.

Objectives

  • To visualize AMR trends across different regions and microorganisms.
  • To offer accessible and educational tools for public health professionals and students.
  • To promote awareness of AMR as a global health concern.

Data Sources

The data used in this site is sourced from reputable global health organizations and research groups including the WHO, . Each CSV file represents AMR statistics for a specific microorganism, broken down by country.

Built With

  • HTML/CSS for page structure and design
  • D3.js for interactive map visualizations
  • Chart.js for generating charts
